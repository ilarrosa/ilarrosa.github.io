Invierte en triángulo equilátero mediante complejos

Esta construcción es puramente algebraica. El valor w se define directamente como w = 1/2 + sqrt(3) i/2, y los puntos A, B y C se entran con ratón y en Propiedades-->Álgebra-->Coordenadas se definen como complejos. Los valores z y z' se calculan a partir de los anteriores:

z = -(B C + A B w + A C w²) / (A + C w + B w²)

z' = -(B C + A C w + A B w²) / (A + B w + C w²)
